# Automate the Job Fetching from Greenhouse, Lever


nodejs version - v21.5.0

## using pnpm instead npm 
1. pnpm init - creates a package.json to manage dependencies
2. pnpm install express - install express for routing
3. pnpm install cors - enable cross-origin requests securely(CORS allows servers to specify who can access their resources)
4. pnpm install mongoose - Object Data Modeling(ODM) for MongoDb


## Setup instructions

1. Clone the repository
2. Run `pnpm install` to install all the dependencies locally
3. Run `docker compose up` to start any dependencies
4. run `npm start` or `pnpm start` to start the server
